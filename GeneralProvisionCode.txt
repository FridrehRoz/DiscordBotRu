Сокращения в коде:
    cmd - command
    ctx - context
    inter - interaction
    info - information
    obj - object
    pbar - progress bar

Термины в коде:
    cog - ког = винтик (Набор команд в подобие модуля)
    guild - гильдия = сервер
    ~ в комментариях означает пример вводных данных


Импорт в коде (иерархия):
    """
    '#' - Заголовок импорта
    '-' - Подстройка к заголовку
    '\' - Иерархия заголовка
    """
    #Встроенные модули (Стандартный пул Python)
        \Общий импорт
        \Детальный импорт
        -if __name__ __main__ (Отладка инициализации)

    #Основные внешние модули (~disnake)
        \Главенствующие модули
        \Второстепенные модули
        \Написанные модули из проекта (Особые)

    #Детальный импорт внешних модулей (~from disnake import Intents)
        \Основные (Общая направленность использования)
        \Вторичные (Узконаправленные)

    #Вспомогательные со программные модули (составляющие код бота)
        \Основные модули (самодостаточный модуль)
        \Ресурсы (требуемые куски кода)

    #Модули отладки (для разработчика)


Наименование пакетов:
    Пакет ресурсов:
        Предмет-пакета_res (Область ответственности)
            -Предмет-пакета_назначение

    Пакет скриптов:
        Назначение-модуля_scripts (Для чего)
            -Назначение-скрипта

    Иные пакеты являются уникальными, вследствие чего
    их названия определяются при разработке.
