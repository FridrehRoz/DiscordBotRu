Сокращения в коде:
    cmd - command
    ctx - context
    inter - interaction
    info - information
    obj - object

Термины в коде:
    cog - ког = винтик (Набор команд в подобие модуля)
    guild - гильдия = сервер
    ~ в комментариях означает пример вводных данных

Импорт в коде:
    #Встроенные модули
    - if __name__ __main__ (Отладка инициализации)
    #Основные внешние модули
    #Детальный импорт внешних модулей
    #Вспомогательные со программные модули (с сопутствующим кодом)
        -Основные
        -Ресурсы
    #Модули отладки (вне работы программы)

Наименование модулей пакетов:
    Пакеты ресурсов:
        Предмет_пакета
            -Предмет_пакета_назначение

    Пакеты скриптов:
        Назначение_модулей_пакета
            -Назначение_модуля
